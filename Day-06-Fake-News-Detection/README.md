# Fake News Detection System

A Machine Learning and NLP project that classifies news articles as REAL or FAKE.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* Matplotlib
* NLP

## Features

* Fake News Detection
* Natural Language Processing
* Text Classification
* Model Saving using Pickle
* Flask Web Application
* Data Visualization

## Project Structure

* data
* models
* src
* templates
* static
* app.py
* requirements.txt

## Workflow

News Article
→ Text Vectorization
→ Machine Learning Model
→ Prediction
→ REAL or FAKE

## Example

Input:

Aliens landed in Hyderabad today

Output:

FAKE

Input:

Government launches new education policy

Output:

REAL
