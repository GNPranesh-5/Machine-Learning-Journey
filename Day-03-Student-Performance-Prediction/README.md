# Student Performance Prediction

A Machine Learning project that predicts student exam scores based on:

* Study Hours
* Attendance Percentage
* Assignments Completed

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* Matplotlib
* Pickle

## Features

* Student Score Prediction
* Data Visualization
* Model Training
* Model Saving using Pickle
* Flask Web Application

## Project Structure

* data
* models
* src
* templates
* app.py
* requirements.txt

## Workflow

Dataset
→ Data Analysis
→ Model Training
→ Prediction
→ Flask Web App

## Example

Input:

Study Hours: 8

Attendance: 90

Assignments: 9

Output:

Predicted Score: 88+
