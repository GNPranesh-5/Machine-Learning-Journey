# Spam Detection System

A Machine Learning and NLP project that classifies messages as Spam or Ham (Not Spam).

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* CountVectorizer
* Logistic Regression

## Features

* Spam Message Detection
* NLP Text Processing
* Model Saving using Pickle
* Flask Web Application

## Example

Input:

Congratulations! You won a free iPhone

Output:

spam
