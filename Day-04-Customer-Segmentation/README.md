# Customer Segmentation using K-Means Clustering

A Machine Learning project that groups customers into different segments based on their Income and Spending Score.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib

## Concepts Learned

* Unsupervised Learning
* K-Means Clustering
* Customer Segmentation
* Data Visualization

## Features

* Customer Grouping
* K-Means Clustering
* Scatter Plot Visualization
* Cluster Analysis

## Project Structure

* data
* src
* README.md
* requirements.txt

## Dataset Features

* Income
* SpendingScore

## Workflow

Dataset
→ K-Means Clustering
→ Cluster Assignment
→ Visualization

## Output

The model automatically groups customers into clusters based on similar behavior patterns.
