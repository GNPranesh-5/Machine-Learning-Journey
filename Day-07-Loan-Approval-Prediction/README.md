# Loan Approval Prediction

A Machine Learning project that predicts whether a loan application is likely to be approved based on applicant information.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Flask
* Matplotlib

## Features

* Loan Approval Prediction
* Logistic Regression Model
* Data Visualization
* Flask Web Application
* User Input Based Predictions

## Dataset Features

* Income
* Credit Score
* Loan Amount

## Workflow

Applicant Data
→ Machine Learning Model
→ Loan Approval Prediction

## Example

Input:

Income: 60000

Credit Score: 750

Loan Amount: 200000

Output:

Loan Approved
