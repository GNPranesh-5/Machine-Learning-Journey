# Movie Recommendation System

A Machine Learning project that recommends movies based on the selected movie's genre.

## Project Overview

This system suggests similar movies to users based on genre matching. If a user enters a movie name, the system recommends other movies from the same genre.

## Technologies Used

* Python
* Pandas
* Flask
* Matplotlib
* HTML
* CSS

## Features

* Movie Search
* Genre-Based Recommendations
* Data Visualization
